☁ Adresa serveru pro ukládání
Zadej URL Render.com serveru. Uloží se do prohlížeče — stačí nastavit jednou na každém zařízení.
☁ Cloudové úložiště — Supabase
Nastavte jednou — data se pak ukládají automaticky při každé změně.
Nastavení (zdarma, 2 minuty):
1. Jdi na supabase.com → New project
2. Settings → API → zkopíruj Project URL a anon public klíč
3. SQL Editor → spusť:
CREATE TABLE rbd_data (key TEXT PRIMARY KEY, value JSONB, ts TIMESTAMPTZ DEFAULT now());
